# 🚀 Resumize

Resumize is an AI-powered resume analysis platform that helps users improve their resumes for ATS systems and recruiters. Upload your resume and get instant AI feedback, ATS scoring, keyword optimization, skill gap analysis, and improvement suggestions.

## ✨ Features

* ATS Resume Scoring (0–100)
* AI-powered resume feedback
* Keyword optimization
* Skill gap detection
* Grammar and clarity suggestions
* Resume improvement tips
* Modern SaaS-style UI
* Responsive design (mobile + desktop)
* Dark / Light mode support

## 🛠️ Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite
* OpenAI API (or any AI model integration)

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/itsvawltt-cell/resumize.github-io.git
```

### Move into Project

```bash
cd resumize.github-io
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will start at:

```
http://localhost:3000
```

## 🏗️ Build for Production

```bash
npm run build
```

Production files will be generated in the `dist/` folder.

## 🚀 Deploy (GitHub Pages)

After building:

```bash
npm run build
```

Then push `dist` contents to GitHub Pages branch or repo.
